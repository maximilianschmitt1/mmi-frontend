# installation

* [node.js](https://nodejs.org/) installieren
* `git clone https://github.com/maximilianschmitt1/mmi-frontend.git`
* `cd mmi-frontend`
* `npm install`

# entwicklung starten

* `npm start`
* [localhost:3000](http://localhost:3000) im Browser aufrufen
